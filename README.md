## 🏢 Organization Repositories

This organization contains the core components of the BricoLink university platform (formerly Artisan Booking):

### 1. 🖥️ [artisan-backend-laravel](https://github.com/artisan-project-tmp/artisan-backend-laravel)
The core backend REST API and the initial frontend Web Views.
* **Tech Stack**: Laravel (PHP), Blade Templates, Vanilla CSS.
* **Purpose**: Serves as the central backend logic, routing, and browser-based interface for booking services.

### 2. 🗄️ [artisan-db-oracle](https://github.com/artisan-project-tmp/artisan-db-oracle)
The Oracle SQL+ database schemas and foundation.
* **Tech Stack**: Oracle Database SQL.
* **Purpose**: Contains the raw `schema.sql` documentation defining the rigid relational data structure for Users, Artisans, and Bookings.

### 3. 📚 [artisan-docs](https://github.com/artisan-project-tmp/artisan-docs)
Centralized project guidelines and setup logic.
* **Purpose**: Contains architecture details, environment prerequisites (like installing the `oci8` PHP extension on Fedora Linux), and our strict Git workflow.
